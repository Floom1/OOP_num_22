# Проект: Система управления студенческой конференцией
Основы объектно ориентированного программирования

**Вариант:** 22


## Задания

### 1. Спроектировать классы для моделирования студенческой конференции

«Ежегодно в колледже проводится научно-практическая студенческая конференция. Она разбивается на секции, каждый студент может подать заявку на участие в нескольких секциях. Каждое выступление студента оценивается в баллах каждым членом жюри конференции (по секциям). Баллы суммируются. По окончании конференции издается сборник со статьями». Построить диаграмму классов.

### 2. Разработать визуальное приложение

Разработать визуальное приложение для демонстрации работы с системой классов. Программа должна формировать на экране отчет «Распределение выступающих по отделениям». Способ хранения данных определяется студентом самостоятельно.

## Запуск

### Визуальное приложение (PyQt6)
```bash
python -m frontend.main
```

### Консольная версия
```bash
python -m exe.main
```
